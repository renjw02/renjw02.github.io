IMP

IMP 是一个简单的示例语言

IMP 语言中包含了命令式程序设计语言最重要、最核心的语言要素。

语法

主要关注抽象语法

语法范畴：

整数集 Z, 用 a, b, c 表示整数。

变元集 Var，用 x, y, z 表示变元。

算术表达式集 AExp，用 e 表示算术表达式。

布尔表达式集 BExp，用 p, q 表示布尔表达式。

语句集 Stmt，用 st 表示语句。

语义

状态

算术表达式的语义

算术表达式 e 在状态 s 下的值 $[[e]]_s$ 递归定义如下：

  • $[[c]]_s = c$, 其中 c 为整数
  • $[[x]]_s = s(x)$，其中 x 为变元
  • $[[e1 + e2]]_s = [[e1]]_s + [[e2]]_s$
  • $[[e1 − e2]]_s = [[e1]]_s − [[e2]]_s$
  • $[[e1 ∗ e2]]_s = [[e1]]_s × [[e2]]_s$

布尔表达式的语义

布尔表达式 p 在状态 s 下的值 [[p]]s 递归定义如下：

  • $[[true]]_s = true$

  • $[[false]]_s = false$

  • $[[e1 = e2]]_s = true 当且仅当 [[e1]]_s = [[e2]]_s$

  • $[[e1 ≤ e2]]_s = true 当且仅当 [[e1]]_s ≤ [[e2]]_s$

  • $[[¬p]]_s = true 当且仅当 [[p]]_s = false$

  • $[[p1 ∧ p2]]_s = true$ 当且仅当 $[[p1]]_s$ 和 $[[p2]]_s $都为真
